The REST Area lets you experiment with https REST apis to see the results, and help automatically generate Typescript Interfaces. Today, it supports GET and POST methods.

select BUILD from the toolbar to get started. then:

  • Select GET or POST for the method
  • Enter the URL
  • Add Parameter or Header name value pairs
  • Submit the request
  • Name the Interface and select an Interface prefix for any subtypes
  • Press Generate Interface
  • Use the copy button to copy the results to your sourcefile.
